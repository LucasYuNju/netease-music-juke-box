# netease-music-juke-box

高仿网易云音乐，项目展示了如何用流行的前端技术开发完整的Web应用。（数据API来自[net-player](https://github.com/MagicCube/net-player)）

## 涉及的技术

- ES6/ES7
- jQuery
- LESS
- Gulp
- Webpack
- Flex Style

## 运行
1. 下载repo
2. `npm install`
3. `gulp dev`
4. 访问localhost:8080
